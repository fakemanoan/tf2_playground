How does this work?

Source uses a special unicode character for text chat colours. You can find this in the TF2 source code that was recently released. However due to a bug in the code, it does not properly strip the colour code magic symbols from user inputted text. This means we can use this in-game, no game modifications necesssary.

Any standard hexadecimal colour code is accepted (ie 0xFFFFFF).

I created this page because others do not follow limitations set in-game properly. Those limitations are:

  • You can't enter text longer than 127 characters (including the additions for colour)
  • You can only do this with a prefix infront of your name (Like *DEAD*, (TEAM) or (Spectator))
  • Due to botting, some text characters are stripped from messages, such as new lines, some special unicode characters, and some ASCII symbols.
  • Some community servers have plugins which fixes this problem. Using it in these servers may get you banned.
